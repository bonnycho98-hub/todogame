# Telegram Formatter Redesign

**Date:** 2026-04-23

## Summary

텔레그램 브리핑/상태 메시지의 포맷을 개선한다. 기계적인 리스트 느낌에서 벗어나, 게임의 핵심 철학("상대를 사랑하기 위해 나를 사랑한다")을 상기시켜주는 따뜻한 반말 어조로 바꾼다.

---

## 변경사항

### 1. 고정 철학 문구 추가

모든 브리핑 메시지(아침/저녁) 상단에 고정 문구를 표시한다:

> 언제나 본질은 상대를 사랑하는 거야. 그걸 위해 널 사랑해줘.

`/status`에는 표시하지 않는다.

### 2. 체크박스 기호 변경

| 이전 | 이후 |
|------|------|
| `□` (미완료) | `·` |
| `☑` (완료) | `✓` |

### 3. 버그 수정

**`?` NPC 이름 문제:** `need_id`가 None인 나 자신 퀘스트는 NPC 이름 대신 `?`를 표시했다. `"나"`로 변경한다.

**서브태스크 없는 퀘스트 제목 중복:** `format_briefing`의 npc_quests 루프에서 서브태스크가 없는 경우 퀘스트 제목을 두 번 출력했다 (formatter.py:82-83). 중복 라인을 제거한다.

### 4. 어조 변경

- 헤더: `🎮 *LOVE QUEST — 오늘의 브리핑*` → `🎮 오늘의 브리핑`
- 저녁 헤더: `🌙 *LOVE QUEST — 저녁 브리핑*` → `🌙 아직 남아있어.`
- 상태 헤더: `📊 *STATUS*` → `📊 지금 너의 상태야.`
- 섹션 헤더: `*\[ 나를 사랑하기 \]*` → `나를 사랑하기`, `*\[ 타인을 사랑하기 \]*` → NPC 이름 직접 표시
- 행복 레벨: `행복 레벨:` → `lv.N ■■□□`

## 최종 메시지 예시

### `/brief` (아침)
```
🎮 오늘의 브리핑

언제나 본질은 상대를 사랑하는 거야. 그걸 위해 널 사랑해줘.

나를 사랑하기
  ✓ 데드버그 코어 안정화 운동 1분
  · 허리 요가하기

신정철 본부장
  · 가치토크 리뷰 준비

lv.3 ■■■■■■■□□
```

### `/brief` (저녁)
```
🌙 아직 남아있어.

언제나 본질은 상대를 사랑하는 거야. 그걸 위해 널 사랑해줘.

신정철 본부장
  · 가치토크 리뷰 준비

lv.3 ■■■■■■■□□
```

### `/status`
```
📊 지금 너의 상태야.

나 자신: 40
신정철 본부장: 40

lv.3 ■■■■■■■□□ (80%)
```
