# 루틴 유도 배너 설계

## 개요

대시보드에서 오늘의 루틴 미완료 시 긴급 배너를 표시해 사용자가 루틴을 완료하도록 유도한다.

## 위치

Hero 카드 바로 아래, "오늘의 루틴" 섹션 레이블 위에 삽입.

## 동작

### 표시 조건
- 오늘의 루틴이 1개 이상 미완료인 경우 항상 표시
- 앱 진입 시점부터 표시 (시간 제한 없음)
- 닫기 버튼 없음 — 루틴 완료만이 유일한 해제 수단

### 상태 1: 미완료 (경고)
- 배경: 빨간 그라디언트 (`#c1121f` → `#e63946`)
- 아이콘: 🔥
- 제목: "오늘 루틴 N개 남았어요" (N = 미완료 개수)
- 부제: "스트릭 M일 — 오늘 놓치면 리셋돼요!" (M = 현재 스트릭)
- 우측 버튼: "지금 하기 →" (클릭 시 루틴 섹션으로 스크롤)

### 상태 2: 완료 (칭찬)
- 마지막 루틴 완료 시 경고 배너 → 칭찬 배너로 교체
- 배경: 초록 그라디언트 (`#2d6a4f` → `#40916c`)
- 아이콘: 🎉
- 제목: "오늘 루틴 완료!"
- 부제: "스트릭 M일 달성 · 잘했어요 ✨"
- 3초 후 fade-out 애니메이션으로 사라짐

## 데이터

기존 대시보드 API(`/api/dashboard`)의 `routine_quests` 배열을 그대로 사용.
- 미완료 개수: `routine_quests.filter(q => !q.is_all_done_today).length`
- 현재 스트릭: 기존 `hero-streak` 값 재사용

백엔드 변경 불필요.

## 구현 범위

- `static/index.html`: 배너 HTML 요소 추가
- `static/app.js`: 배너 렌더 함수 + 루틴 완료 시 상태 전환 로직
- `static/style.css`: 배너 스타일 (경고/완료 두 가지 클래스)
